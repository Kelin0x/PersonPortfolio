# 3D翻转瓦片动画展示页面

这是一个具有炫酷3D翻转效果的展示页面，主要包含以下特性：

## 主要功能

1. 3D瓦片翻转效果
   - 6x6网格布局的瓦片板
   - 鼠标悬停触发单个瓦片翻转
   - 全局翻转按钮控制所有瓦片
   - 平滑的动画过渡效果

2. 自定义光标
   - 环形旋转文字效果
   - 跟随鼠标平滑移动
   - 持续旋转动画

3. 背景网格效果
   - 响应式背景方块布局
   - 根据屏幕尺寸自动调整

## 技术栈

- HTML5
- CSS3 (3D变换、动画)
- JavaScript (ES6+)
- GSAP (动画库)
